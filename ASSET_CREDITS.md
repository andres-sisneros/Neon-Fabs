# Asset Credits

Track all third-party art, audio, fonts, and visual packs used by Neon Fabs.

Current build: no third-party visual assets are bundled. The intro cinematic is built with local HTML/CSS.

## Template

```text
Asset:
File:
Author:
Source URL:
License:
Changes:
Notes:
```

## License Preferences

- Best for prototype: CC0/public domain.
- Also usable with care: CC-BY, OGA-BY, MIT/Apache-style asset licenses.
- Avoid for now: CC-BY-SA, GPL/LGPL art, non-commercial licenses, no-derivatives licenses, ambiguous itch.io asset pages, and any asset copied from an existing commercial game.
