# Design Workspace

Use this folder for the human-authored soul of Neon Fabs: lore, tone, names, art direction, factions, city notes, and item concepts.

The intended workflow:

1. Write rough ideas in Markdown here.
2. Promote stable names/flavor into `content/creative-overrides.js`.
3. When a system stabilizes, we can move its data into structured content files.

This keeps creative work readable and preserves a clean boundary between game design, authored lore, and runtime code.
