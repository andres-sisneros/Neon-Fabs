# Assets

Place game art, audio, and downloaded third-party assets here.

## Rules

- Prefer original art or CC0/public-domain assets while the game direction is still changing.
- Every third-party asset needs an entry in `ASSET_CREDITS.md`.
- Keep source URLs and license names with the asset before using it in the UI.
- Avoid GPL art, ShareAlike art, fan art, ripped sprites, and unclear licenses unless we intentionally review the implications.
- Do not add paid assets to the public repo unless the license allows redistribution.

## Suggested Folders

```text
assets/
  city/
  fabs/
  items/
  vehicles/
  ui/
  audio/
```
